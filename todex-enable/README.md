# todex
converts to dex on ps4 using memory spoof

Supported versions are:

FW 5.05
FW 6.50 / 6.51 / 6.70 / 6.72 / 6.72
FW 7.00 / 7.01 / 7.02
FW 7.50 / 7.51 / 7.55
FW 8.00 / 8.01 / 8.03
FW 8.50 / 8.52
FW 9.00
FW 9.03 / 9.04
FW 9.50 / 9.51 / 9.60
FW 10.00 / 10.01
FW 10.50 / 10.70 / 10.71
FW 11.00

How to Building;
make FW=XXXX

ie; make FW=1100

How to cleaning;
make FW=1100 clean

Requires being built with latest the Scene Collective Payload SDK
